# World Cup 2026 Top Trumps

A visual browser game inspired by Top Trumps, using a World Cup 2026-style tournament flow.

## Game Rules

- Every team has exactly **20 points** split across:
  - `defensive`
  - `midfield`
  - `attack`
  - `coolness`
- Fixtures run through:
  - Group stage (12 groups x 4 teams)
  - Round of 32
  - Round of 16
  - Quarter Finals
  - Semi Finals
  - Third Place
  - Final
- For each match, a random category is chosen.
- The team with the higher value in that category wins.

## Run Locally

Open `index.html` in your browser.

## Play Online (GitHub Pages)

1. Push this folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set source to your main branch and root folder.
4. Save and open the generated Pages URL.

No build step required.
